# Codebase Stack: Nexus-OS

## Frontend Stack

| Layer | Technology | Version | Rationale |
|---|---|---|---|
| **Framework** | Next.js | 16.2.1 | Modern App Router, SSR, and excellent performance. |
| **Library** | React | 19.2.4 | Latest stable React with enhanced features. |
| **Language** | TypeScript | 5.x | Strong typing for better developer experience and reliability. |
| **Styling** | Tailwind CSS | 4.x | Utility-first CSS with next-gen engine (v4). |
| **UI Icons** | Lucide React | 1.7.0 | Clean, consistent, and performant icon set. |
| **Markdown** | react-markdown | 10.1.0 | Robust markdown rendering in React. |

## Backend Stack (Planned/Under Development)

| Layer | Technology | Version | Rationale |
|---|---|---|---|
| **Language** | Java | 21 | Long Term Support (LTS) version with modern features. |
| **Framework** | Spring Boot | 3.x | Mature, powerful backend framework with huge ecosystem. |
| **Database** | PostgreSQL | 16 | Relational database with strong JSONB support for docs. |
| **Security** | Spring Security | 3.x | Industry standard for authentication and authorization. |

## Build & Deployment

- **Package Manager:** npm (frontend) / Maven (backend - planned).
- **Orchestration:** Docker Compose (planned for unified deployment).
- **Editor:** VS Code / Cursor recommended.
